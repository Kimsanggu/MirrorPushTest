# ObjectFinder

How To Use

1. Add NullFinder Script to Any GameObject;
2. Press FindButton;

--------------------------------------------------------

 * 씬에서 버튼에 missing or null 값이 들어가 있거나, Image 컴포넌트에 source가 missing or null 인 경우를 찾아준다.

1. NullFinder 에디터 스크립트를 하이라키 안에 있는 오브젝트에 추가한다.
2. Find 버튼을 누른다.
